# Computer Science Ecosystem Project

## Project Description
This project was developed for the Computer Science Ecosystem course to demonstrate fundamental skills in web development, terminal programming, and shell scripting. The project showcases core technical talents through an interactive personal resume website and a terminal application, organized and managed using Git and GitHub.

## Instructions for Running the Project Locally

Follow these steps to download and run the ecosystem components on your machine:

### 1. Download the Project
*   Download the project repository as a **ZIP file** from GitHub.
*   Extract the ZIP folder onto your local computer.

### 2. Open the Project in VS Code
*   Launch **Visual Studio Code**.
*   Go to `File > Open Folder...` and select the extracted project directory.

### 3. Run the Terminal Application
*   Locate and open the file named `vending_machine.py` in the VS Code sidebar explorer.
*   Run the script using the Python terminal interpreter inside VS Code.

### 4. Launch the Interactive Website
*   Navigate to your HTML file (`index.html` or `projectcv.html`) in the editor sidebar.
*   Right-click anywhere inside the HTML code.
*   Select **"Open with Live Server"** to view and interact with the website in your web browser.

### 5. Run the Asset Analysis Script (Optional)
*   Open your terminal inside the project directory and execute the bash script to count your HTML tags:
    ```bash
    bash ayyan_bash_script.sh
    ```
